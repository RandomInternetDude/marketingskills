# Brand Kit & Description Variants

Everything a submission needs, assembled once and reused everywhere. The brand **facts** stay identical on every platform; the **descriptions** vary per surface so nothing reads as duplicated boilerplate.

Swap `[site]`, `[category]`, `[audience]`, `[one-liner]`, and `[differentiator]` with real values — ideally pulled from `.agents/product-marketing.md`.

---

## Rule 0 — Lock the canonical brand kit first

Fill this in before the first submission. Every form answer comes from here; an agent filling forms should never improvise a field.

```
Site name:      [Exact brand name — one spelling, one capitalization]
Canonical URL:  https://[domain]/          (decide www vs non-www, trailing slash)
One-liner:      [What the site is, in one sentence]
Category:       [Primary category as platforms name it — e.g., "Education", "Career Resources"]
Founding year:  [YYYY]
Contact email:  listings@[domain]          (dedicated alias — directories generate spam)
Business name:  [Legal/DBA entity if different from site name]
Address:        [Registered business / virtual office — NEVER a home address; blank if none]
Phone:          [Business line if any; blank if none]
Social handles: [X] · [Facebook] · [LinkedIn] · [YouTube] · [Pinterest] · [Instagram]
Logo:           PNG + SVG + square 1024×1024 + favicon
Images:         2–3 representative screenshots (homepage, best content)
```

**Consistency rule:** name, URL, category, and address are byte-for-byte identical on every platform. Only descriptions and tags vary.

---

## The Description Ladder

Prepare four lengths once, then vary the *opening sentence and emphasis* per surface (next section). Platforms ask for one of these four almost universally.

**Tagline (under 10 words):**
> [Outcome] for [audience].
> *Example: "Everything you need to become a CNA, in one place."*

**Short (~60 chars):**
> [Site] — [category noun] for [audience].

**Standard (~150 words):**
> [Site] is a [category] that helps [audience] [core outcome]. [2–3 sentences on what's actually there: the guides, tools, data, or listings and what makes them useful]. [1 sentence of differentiation — what it covers that alternatives don't, or how it's organized]. [1 sentence on who it's for and the primary call to action].

**Extended (~300 words):**
> The standard description plus: origin/why it exists (1–2 sentences), the main content areas as a short list, and any proof points (years running, volume of resources, audience size — only real numbers).

## Variation Rules by Surface Type

| Surface | Open with | Emphasize | Avoid |
|---|---|---|---|
| **Identity profiles** (LinkedIn, Crunchbase) | Factual one-liner | Completeness — every field filled, real founding year, real category | Marketing adjectives |
| **Web directories** (Hotfrog, Cylex, WebWiki) | Category-first sentence ("[Site] is a [category] resource…") | Correct categorization + clean NAP/URL | Keyword stuffing the name field |
| **Content platforms** (Medium, Substack, Tumblr) | Editorial voice — the topic, not the brand | What readers get from the content | Corporate boilerplate |
| **Blog directories** (Feedspot, Alltop) | The blog's beat in one line | Topic clarity + posting cadence | Overselling frequency you won't keep |
| **Bookmarking** (Scoop.it, Pearltrees) | The page being curated, not the site | Why this page is worth a reader's click | Bookmarking every URL you own |
| **Niche directories** | Fit with the niche in the first sentence | The exact audience served | Forcing a listing where it doesn't belong |

**Tag rotation:** build a pool of 12–15 tags, use 5–8 per platform, never the identical set twice in the same tier.

**Tag pool starter (education/career content site):**
[category], [niche] training, [niche] certification, career resources, education, career change, [audience], study guides, practice tests, [state/city if geo-relevant], free resources, how to become a [role]

## Anchor Text Rule

Directory and profile links should use **brand-name or bare-URL anchors** ("[Site]", "[domain].com") — never exact-match keyword anchors ("best CNA classes"). Foundation links with commercial anchors are a classic manipulation footprint. Where a platform lets you choose anchor text, choose the brand.

## Appendix — Local NAP Block (Tier 7 only)

Only for a business with a genuine physical location. Lock this before any local submission; byte-for-byte identical everywhere (abbreviations, suite formatting, and phone format all count).

```
Name:     [Exact legal/DBA name — no keywords added]
Address:  [Street, Suite] · [City], [ST] [ZIP]
Phone:    ([XXX]) XXX-XXXX   (local number preferred)
Website:  https://[canonical-domain]/[location page if multi-location]
Hours:    [consistent weekly hours]
Primary category:     [precise — e.g., a school: "Nursing school" / "Vocational school" / "Adult education school"]
Secondary categories: [as the platform allows]
```

**Google Business Profile description (750 chars) skeleton:**
> [Business] is a [category] serving [city] and [surrounding area]. [2–3 sentences: what you offer, what makes it credible — approvals, years, outcomes with real numbers]. [Practical details: schedule/availability, how to start]. Call [phone] or visit [url].

**Review request (send once, the week the customer succeeds):**
> Hi [name] — congratulations on [their outcome]. If you have 60 seconds, would you share your experience for others deciding where to [do the thing]? Direct link: [Google review link]. Your honest take helps someone else take the same step.

Rules: one direct link, one follow-up max, never incentivize Google/Yelp reviews, never solicit on Yelp specifically, respond to every review.

## Do / Don't Quick Reference

**DO:**
- Lock the brand kit first; agents fill forms from it verbatim.
- Vary the description opener per surface type.
- Use real numbers where you have them; plain language where you don't.
- Use brand-name anchors everywhere.
- Keep a per-platform quirks note (char limits, category names) in the tracker.

**DON'T:**
- Paste the identical 150-word description on every platform.
- Put a home address on anything public.
- Stuff keywords into name fields (rejection on directories; suspension on Google).
- Claim numbers, approvals, or affiliations you can't verify.
- Create more profiles than you'll maintain.
